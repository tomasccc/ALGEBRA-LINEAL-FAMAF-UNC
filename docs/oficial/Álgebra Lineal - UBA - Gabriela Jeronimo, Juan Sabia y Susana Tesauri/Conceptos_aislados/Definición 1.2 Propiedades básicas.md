Sea $* : A \times A \to A$ una operación.
- $*$ se dice **asociativa** si $(a * b) * c = a * (b * c)$  $\forall a, b, c \in A.$
- Se dice que $*$ tiene **elemento neutro** si $\exists e \in A$ tal que $e * a = a * e = a$ para cada $a \in A$.
  - (**Observación**: que si $*$ tiene elemento neutro, este es único, ya que si $e'$ son elementos neutros, $e' = e * e' = e$).
- Si $*$ tiene elemento neutro $e$, se dice que todo elemento tiene **inverso** para $*$ si $\forall a \in A, \exists a' \in A$ tal que $a * a' = a' * a = e$.
- $*$ se dice **conmutativa** si $a * b = b * a \forall a, b \in A$.
****
Se suele tener en cuenta otra propiedad a la hora de analizar conjuntos.
- **Cerradura.** para todo $a,b∈A$, el resultado $a∗b$ también pertenece a $A$. En la notación $∗:A×A→A$, la cerradura está implícitamente indicada.
