**Teorema 1.24** Sea $H$ un sistema lineal homogéneo de $n$ ecuaciones y $n$ incógnitas. Sea $H$ un sistema equivalente a $H$ cuya matriz $B$ es triangular superior. Entonces $H$ tiene solución única si y sólo si $B_{ii} \neq 0 \quad \forall 1 \leq i \leq n$. 
****
**Demostración.** $(\Leftarrow)$ Supongamos que $B = \begin{pmatrix} B_{11} & \cdots & B_{1n} \\ 0 & \ddots & \vdots \\ 0 & 0 & B_{nn} \end{pmatrix}$ con $B_{ii} \neq 0 \quad \forall 1 \leq i \leq n$.

Entonces, la última ecuación del sistema $H'$ es $B_{nn}x_n = 0$ y, como $B_{nn} \neq 0$, resulta que $x_n = 0$. Reemplazando en la ecuación anterior $x_n$ por $0$, queda $B_{n-1,n-1}x_{n-1} = 0$, de donde $x_{n-1} = 0$.
Siguiendo de este modo, para cada $k = n - 2, \ldots, 1$ de la $k$-ésima ecuación se obtiene $x_k = 0$.

$(\Rightarrow)$ Ahora de forma no trivial, se muestra que si algún pivote es cero, entonces aparece una solución no trivial (no todos ceros).
![[Pasted image 20250911181034.png]]
1. Suponemos que $B_{11} \neq 0$, $B_{22} \neq 0, \ldots, B_{ii} \neq 0$, pero en la posición siguiente aparece un cero en la diagonal:
$$B_{i+1,i+1} = 0$$
2. La matriz triangular queda con las primeras $i$ filas “bien definidas” y la fila $i + 1$ hacia abajo degenerada. Eso provoca que haya una columna sin pivote.
3. Entonces **elegimos** **una** **solución** **particular** sencilla:
$$x_{i+1} = 1,\quad x_{i+2} = \cdots = x_n = 0$$
Esa es una elección libre, posible porque no hay pivote en esa columna.
4. A partir de ahí, van hacia atrás usando las ecuaciones con $B_{ii} \neq 0$ para despejar las demás variables:$$x_i = \frac{-B_{i,i+1}}{B_{ii}},\quad x_{i-1} = \cdots$$Y así sucesivamente se calculan $x_1, \ldots, x_i$.
5. Resultado: se construye una solución del tipo
$$(x_1, \ldots, x_i, 1, 0, \ldots, 0)$$Como hay una elección libre para $x_{i+1}$, el sistema tiene infinitas soluciones, y podemos ir calculando cada $x_i$ en función del valor $x_{i+1}$ que elijamos. Esto muestra que $B_{ii} = 0$ para que se cumpla el enunciado.
