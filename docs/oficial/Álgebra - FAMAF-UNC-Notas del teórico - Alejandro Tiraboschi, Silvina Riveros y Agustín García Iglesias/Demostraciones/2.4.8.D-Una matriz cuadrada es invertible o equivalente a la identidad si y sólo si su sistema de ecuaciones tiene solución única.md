#### **Teorema 2.4.8.** Sea $A$ una matriz $n \times n$. Entonces, $A$ es equivalente por filas a la matriz $I_n$ si y sólo si el sistema $AX = Y$ tiene una única solución.

($⇒$) Suponemos que $A \sim I_n$. Eso significa que resolver $AX = Y$ es equivalente a resolver: $I_n X = Z$, para algún vector $Z$ obtenido después de aplicar las operaciones de filas.
La matriz identidad tiene $1$ en la diagonal y $0$ en las demás posiciones. Entonces el sistema que resulta es:$$x_1 = z_1,\space x_2 = z_2,\space \ldots \space, x_n = z_n$$Este sistema fija de manera única cada variable, y entonces hay una única solución.

($⇐$) Suponemos ahora que el sistema $AX = Y$ siempre tiene una única solución. 
Sea $R$ la matriz reducida por filas (**MERF**) de $A$. Resolver $AX = Y$ es equivalente a resolver $RX = Z$, para algún $Z$ (recordemos [[2.4.1.D-Toda matriz es equivalente por fila a una matriz escalón reducida por fila]]).  
Como por hipótesis siempre hay una única solución, no pueden existir variables libres. Eso significa que cada columna tiene pivote, y entonces el número de pivotes es $n$. Es decir, $R$ tiene $n$ filas no nulas, lo cual implica que ninguna fila de $R$ es nula. En ese caso, la única **MERF** posible es la identidad $R = I_n$. 
Como $A$ y $R$ son equivalentes por filas, concluimos que $A \sim I_n$.
