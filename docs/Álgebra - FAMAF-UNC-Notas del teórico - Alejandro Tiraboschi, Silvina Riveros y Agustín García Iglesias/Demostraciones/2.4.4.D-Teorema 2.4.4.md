#### **Sea $AX = Y$ un sistema de $m$ ecuaciones lineales y $n$ incógnitas con coeficientes en $K$. Entonces...
1. El sistema homogéneo $AX = 0$ bien tiene a $0$ como única solución, o bien tiene infinitas soluciones.
2. Si $Y \neq 0$, entonces el sistema o bien no tiene solución, o bien tiene una solución, o bien tiene infinitas soluciones.
****
###### **Demostración**
Consideremos el sistema homogéneo  
$$AX = 0.$$
Siempre existe al menos una solución: el vector $X = 0$, porque al multiplicar se cumple \($A0 = 0$\).  
- Si **no hay variables libres**, es decir, si todas las incógnitas se determinan de manera única en el proceso de reducción, entonces la única solución es  
$$x_1 = x_2 = \cdots = x_n = 0.$$
- Si en cambio **sí hay variables libres**, entonces esas variables pueden tomar infinitos valores, lo cual produce **infinitas soluciones**.  
Por lo tanto, el sistema homogéneo tiene o bien **una única solución (la trivial)**, o bien **infinitas soluciones**.  

Ahora supongamos que el sistema no es homogéneo:  
$$AX = Y, \quad Y \neq 0.$$
Las soluciones, si existen, también se describen con la misma forma general obtenida al resolver mediante reducción.  
- Si **no aparecen variables libres**, entonces el sistema tiene a lo sumo **una única solución**.  
- Si **aparecen variables libres**, entonces el sistema tiene **infinitas soluciones**.  
- Y si el sistema es **incompatible** (es decir, durante la reducción aparece una ecuación imposible como \($0 = 1$), entonces el sistema **no tiene solución**.  $\Box$
