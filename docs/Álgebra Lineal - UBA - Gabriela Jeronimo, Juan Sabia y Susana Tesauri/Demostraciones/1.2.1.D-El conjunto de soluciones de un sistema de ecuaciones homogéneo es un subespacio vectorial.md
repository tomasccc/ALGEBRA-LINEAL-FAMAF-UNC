**Observación 1.17.** El conjunto de las soluciones de un sistema lineal homogéneo con $m$ incógnitas es un subespacio de $K^m$ .
****
**Demostración.** 
$$S = \{ (x_1, \ldots, x_n) \in K^n : \begin{cases} a_{11}x_1 + \cdots + a_{1n}x_n = 0 \\ \vdots \\ a_{m1}x_1 + \cdots + a_{mn}x_n = 0 \end{cases} \}$$
es un subespacio de $K^n$, pues $S = \bigcap_{i=1}^m S_i$, donde $$S_i = \{ (x_1, \ldots, x_n) \in K^n : a_{i1}x_1 + \cdots + a_{in}x_n = 0 \}\quad (1 \leq i \leq m)$$ y cada $S_i$ es un subespacio de $K^n$, debido a que **[[1.1.3.D-La intersección de subespacios es subespacio]]**.
